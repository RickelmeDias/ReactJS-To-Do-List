# To-Do List with React

```
Challenge: Practicing Concepts
```

#### Project:

To create functionalities to a to-do list website using React and Typescript:

- Add new Task
- Check/Uncheck the Task as finished
- Delete Task
- Show task completion progress

###### Knowledges used:

- States
- Immutability
- Arrays, Lists, Key, Filter, Reduce, Map
- Props
- Components

#### How to run:

1. Clone this repostiory;
2. Run `npm install` to install the dependencies;
3. Run `npm run dev` to run the project;
4. When finish the previous commands, open the *localhost url* on the console log and use the application.
